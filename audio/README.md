# 音频文件

请从免费来源下载音频，放到这个目录。

## 背景音乐

- `main-loop.mp3` — 主循环（暗色 ambient / chiptune，1-3分钟循环）✅ 已有
- `explore.mp3` — 地图探索用（可选，如果只有一首就只用 main-loop）
- `camp.mp3` — 营地/对话用（可选，温暖一点的氛围）

## 环境音效（新增）

- `amb-crickets.mp3` — 虫鸣/蟋蟀声（森林夜间氛围）
- `amb-campfire.mp3` — 篝火噼啪声
- `amb-rain.mp3` — 雨声
- `amb-wind.mp3` — 风声

环境音会循环播放，可以叠加多个（比如虫鸣+篝火）。

## 推荐下载来源（全部免费商用）

1. Pixabay Music: https://pixabay.com/music/ + https://pixabay.com/sound-effects/
   - 音乐搜 "dark ambient loop" 或 "pixel game calm"
   - 音效搜 "crickets loop" "campfire" "rain ambience" "wind"

2. Freesound: https://freesound.org/
   - 搜 "crickets night loop" "fire crackling" "rain loop" "gentle wind"
   - 筛选 CC0 协议

3. OpenGameArt: https://opengameart.org/
   - 搜 "ambient loop"，筛选 CC0 协议

## 格式要求

- MP3 格式
- 时长 1-3 分钟为佳（会循环播放）
- 文件不要太大（< 3MB），网页加载需要
- 环境音建议时长 30秒-2分钟，无明显起止感（方便无缝循环）
