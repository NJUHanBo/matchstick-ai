# 背景音乐文件

请从以下来源下载免费音乐，放到这个目录：

## 需要的文件

- `main-loop.mp3` — 主循环（暗色 ambient / chiptune，1-3分钟循环）
- `explore.mp3` — 地图探索用（可选，如果只有一首就只用 main-loop）
- `camp.mp3` — 营地/对话用（可选，温暖一点的氛围）

## 推荐下载来源（全部免费商用，无需署名）

1. Pixabay Music（最推荐）: https://pixabay.com/music/
   - 搜 "dark ambient loop" 或 "pixel game calm" 或 "chiptune melancholic"
   - 下载后重命名为上述文件名

2. OpenGameArt: https://opengameart.org/
   - 搜 "ambient loop"，筛选 CC0 协议
   - 推荐："Contemplation"（soft synth loop）
   - 推荐："Dark Cavern Ambient"（暗色氛围）

3. FreePD: https://freepd.com/
   - 全部 CC0，点击即下载

## 格式要求

- MP3 或 OGG 格式
- 时长 1-3 分钟为佳（会循环播放）
- 文件不要太大（< 5MB），网页加载需要
